RESTful API Documentation Lab 131

This document explains the use of the RESTful API for front-end developers of a Web server and how to use the API.


The fields that this API uses

  • id - A uniqe item ID
  • author - Author's name in string
  • image - Image url of an author's photo
  • alt - Alt text for the photo in string
  • description - Description of the photo in string
  • Tags - Tags in string (or array of strings)

Create

The Create functionality allows you to add data for a new photo item.

Endpoint URI

The endpoint URI for creating a new photo item is: /x

HTTP Method

The HTTP method for creating a new photo item is POST.

HTTP Response Codes

  • 201 Created - The item creation is successfull, the photo is addded.
  • 400 Bad Request - There was a prronlem with the request data.

MIME Format

The resource is returned in application/json format in the HTTP response message. Look below for an example.

Example

Input:

	  POST /x
	  Content-Type: application/json
	  {
	    "author": "New image",
	    "image": "https://image.com/photo.png"
	    "alt": "alt-text"
	    "description": "This is a description for the photo"
	    "tags": "tags, photo"
	  }

Output:

	  HTTP/1.1 201 Created
	  Content-Type: application/json
	  {
	    "id": 1,
	    "author": "New image",
	    "image": "https://example.com/photos/1.jpg"
	    "alt": "alt-text"
	    "description": "This is a description for the photo"
	    "tags": "tags, photo"
	  }

Retrieve

The Retrieve functionality allows you to get and list the data of a specific item.

Endpoint URI

The endpoint URI for retrieving a photo item is: /x/{id}, where {id} is the ID of the photo item to get.

HTTP Method

The HTTP method for retrieving a photo item is GET.

HTTP Response Codes

  • 200 OK - The photo item was successfully retrieved.
  • 404 Not Found - The photo item with the specified ID was not found.

MIME Format

The resource is returned in application/json format in the HTTP response message. Look below for an example.

Example

Input:

	  GET /x/1

Output:

	  HTTP/1.1 200 Photo Details Retrieved
	  Content-Type: application/json
	  {
	    "id": 1,
	    "author": "New image",
	    "image": "https://example.com/photos/1.jpg"
	    "alt": "alt-text"
	    "description": "This is a description for the photo"
	    "tags": "tags, photo"
	  }

Update

The Update functionality allows you to change data of an item in the database.

Endpoint URI

The endpoint URI for updating a photo item is: /x/{id}, where {id} is the ID of the item to update.

HTTP Method

The HTTP method for updating a photo item is PUT.

HTTP Response Codes

  • 200 OK - The item was successfully updated.
  • 400 Bad Request - There was an issue with the requested data.
  • 404 Not Found - The photo item with the given ID was not found.

MIME Format

The resource is returned in application/json format in the HTTP response message.

Example

Input:

	  PUT /x/1
	  Content-Type: application/json
	  {
	    "id": 1,
	    "author": "Updated image",
	    "image": "https://example.com/updated-photo.jpg"
	    "alt": "updated-alt-text"
	    "description": "This is an updated description for the photo"
	    "tags": "updated, tags, photo"
	  }

Output:

	  HTTP/1.1 200 OK
	  Content-Type: application/json
	  {
	    "id": 1,
	    "author": "Updated image",
	    "image": "https://example.com/updated-photo.jpg"
	    "alt": "updated-alt-text"
	    "description": "This is an updated description for the photo"
	    "tags": "updated, tags, photo"
	  }

Delete

The Delete functionality allows you to remove data from the database of a specific item.

Endpoint URI

The endpoint URI for deleting a photo item is: /x/{id}, where {id} is the ID of the photo item to delete.

HTTP Method

The HTTP method for deleting a photo item is DELETE.

HTTP Response Codes

  • 204 No Content - The photo item was successfully deleted.
  • 404 Not Found - The photo item with the specified ID was not found.

MIME Format

Only the HTTP response code is returned in the the delete function.

Example

Input:

	  DELETE /x/1

Output:

	  HTTP/1.1 204 Item Deleted
